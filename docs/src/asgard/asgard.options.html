asgard.options package¶

Module contents¶

asgard.options.get_option(namespace, option_name)[código fonte]¶

Gets an options based on an namespace and an option name. All option are read from Environ variables. The formation name is: HOLLOWMAN_<NAMESPACE>_<OPTIONNAME> If OPTIONNAME is a multi-value (list) option a numver suffix can be used, eg:

HOLLOWMAN_<NAMESPACE>_<OPTIONNAME>_<INDEX>

Name

Name of the filter who owns this option

Option_name

Name of the option

Retorna

The option value, always as a list of values
